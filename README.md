# Muqi Tech — emuqi.com

Brand site for MUQI Tech (山东木齐健康科技有限公司).

**Live at**: https://mqtech-martin.github.io/emuqi/ (GitHub Pages)

## Structure

```
├── index.html      # Homepage
├── about.html      # About Martin Chen
├── projects.html   # Projects showcase
├── style.css       # Global styles
├── script.js       # Interactions
└── .github/workflows/deploy.yml  # Auto-deploy
```

## Tech Stack

- GitHub Pages (free static hosting)
- Pure HTML/CSS/JS (no frameworks)
- Ghost CMS (blog.emuqi.com — Blog & Newsletter)

## Deploy

Push to `main` → GitHub Actions auto-deploys to GitHub Pages.
