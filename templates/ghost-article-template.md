# Ghost Article Template (blog.emuqi.com)

Use this template when preparing content for Ghost CMS.

---

## YAML Frontmatter (Ghost metadata)

```
Title: Your Article Title Here
Slug: your-article-title-here
Feature Image: /content/images/2026/xx/your-image.jpg
Excerpt: One-sentence summary of the article (for SEO & cards)
Tags: Hydrogen, AI, Manufacturing, Health
Meta Title: Keyword-rich SEO title (max 60 chars)
Meta Description: SEO description (max 160 chars)
Og Image: /content/images/2026/xx/social-card.jpg
```

---

## Content Structure

### Section 1: Hook (2-3 sentences)

The opening paragraph that grabs attention. State the problem or observation.

### Section 2: Core Content

Use short paragraphs. Include data, quotes, or personal experience.

> Pull quotes are useful for emphasis.

### Section 3: Deep Dive

Break down technical or complex ideas. Use bullet points:
- Point one
- Point two
- Point three

### Section 4: Application / Case Study

Describe real-world application or examples.

### Section 5: Call to Action

What should the reader do next? Visit a page, try a product, subscribe.

## SEO Checklist

- [ ] Meta Title ≤ 60 chars, includes primary keyword
- [ ] Meta Description ≤ 160 chars, compelling
- [ ] Feature Image set (1200×630px recommended)
- [ ] Tags minimum 2, maximum 5
- [ ] Internal links to other articles
- [ ] External links open in new tab
- [ ] Social share card (OG image) set
